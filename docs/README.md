# Documentación interna — sabhoy.es (San Antonio de Benagéber)

Documentación **no pública** de contexto local para redactores de SAB Hoy. Objetivo: conservar memoria histórica, ordenar cronologías y facilitar cobertura futura **sin publicar piezas periodísticas**.

## Cómo usar este directorio

- Priorizar **hechos comprobables**; separar **declaraciones públicas** e **interpretaciones periodísticas**.
- Cada ficha incluye sección **Fuentes** y, cuando procede, **Límites de esta ficha**.
- Enlaces internos con **rutas relativas** desde el documento que enlaza.
- Fechas en formato ISO (`YYYY-MM-DD`) cuando ayuden a ordenar versiones.
- **Enlaces desde noticias publicadas:** consultar [Rutas internas (web pública)](rutas-internas.md) antes de redactar.

## Convenciones de archivo

1. Formato estándar por ficha: Resumen, Cronología, Hechos relevantes, Personas implicadas, Entidades relacionadas, Situación actual, Posibles líneas futuras, Fuentes.
2. No inventar datos: marcar lagunas como «pendiente de verificación».
3. Actualizar la fecha de revisión al pie cuando se modifique una ficha.
