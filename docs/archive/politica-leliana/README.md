# Archivo interno — Política local de l’Eliana

Documentación **no pública** de contexto político municipal: memoria, relaciones entre formaciones, hitos verificables y rutas para ampliar el archivo.

## Cómo usar este directorio

- Objetivo: contextualizar cobertura futura y mantener cronologías sin publicar “piezas SEO”.
- Criterio: priorizar **hechos comprobables**; separar **declaraciones públicas** e **interpretaciones periodísticas**.
- Extensión: añadir nuevos `.md` en la subcarpeta correspondiente y enlazarlo desde aquí y desde [Fuentes](fuentes/politica-local.md).
