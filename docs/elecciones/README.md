# Elecciones municipales San Antonio de Benagéber 2027

## Resumen

Las **elecciones municipales** en SAB están convocadas para el **domingo 23 de mayo de 2027** (calendario electoral estatal). La corporación actual (13 concejales, mandato 2023–2027) llega a la campaña con **dos cambios de alcaldía** en una legislatura (Tejedor 2023–2025; Santafosta desde sept. 2025). sabhoy.es centraliza cobertura en `/elecciones-municipales-sab-2027`.
